# 参与贡献

感谢你愿意帮助 CodePulse 变得更好。

## 提交问题

- Bug 请包含 CodePulse 版本、Windows 版本、复现步骤和预期结果。
- 截图和日志中请先移除路径、用户名、提示词等隐私信息。
- 新功能建议请描述具体使用场景，而不只是控件或实现方式。

## 开发流程

1. Fork 仓库并从 `main` 创建功能分支。
2. 执行 `npm install` 安装依赖。
3. 完成修改后运行 `npm test` 与 `npm run build`。
4. 提交 Pull Request，并说明行为变化和验证方式。

状态探测相关变更应保持只读，不应修改 Codex 的数据库、会话文件或配置。
